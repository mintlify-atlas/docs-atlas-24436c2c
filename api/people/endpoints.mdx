## Person Endpoints

### List Persons

<api-endpoint method="get" path="/api/data/persons">
  Retrieve all persons with optional filtering and pagination.
</api-endpoint>

**Query Parameters:**

<ParamField query="page" type="integer">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="integer">
  Number of results per page
</ParamField>

<ParamField query="relations" type="boolean" default="false">
  Include related entities (departments, etc.)
</ParamField>

<ParamField query="with_pass_hash" type="boolean" default="false">
  Include password hash (admin only)
</ParamField>

**Permissions:** All authenticated users (limited view for non-admins)

**Response:**

```json
[
  {
    "id": "person-uuid",
    "first_name": "Jane",
    "last_name": "Smith",
    "email": "jane.smith@studio.com",
    "role": "user",
    "active": true,
    "departments": ["dept-uuid"],
    "has_avatar": false,
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-15T10:30:00Z"
  }
]
```

---

### Create Person

<api-endpoint method="post" path="/api/data/persons">
  Create a new person with the provided data.
</api-endpoint>

**Permissions:** Admin only

**Request Body:**

<ParamField body="first_name" type="string" required>
  Person's first name
</ParamField>

<ParamField body="last_name" type="string" required>
  Person's last name
</ParamField>

<ParamField body="email" type="string" format="email" required>
  Person's email (can be null for bots)
</ParamField>

<ParamField body="password" type="string">
  Password (will be encrypted)
</ParamField>

<ParamField body="role" type="string" default="user">
  Role: `user`, `admin`, `supervisor`, `manager`, `client`, `vendor`
</ParamField>

<ParamField body="contract_type" type="string" default="open-ended">
  Contract type: `open-ended`, `fixed-term`, `short-term`, `freelance`, `apprentice`, `internship`
</ParamField>

<ParamField body="active" type="boolean" default="true">
  Whether the person is active
</ParamField>

<ParamField body="is_bot" type="boolean" default="false">
  Whether this is a bot/service account
</ParamField>

<ParamField body="departments" type="array">
  Array of department IDs
</ParamField>

<ParamField body="expiration_date" type="string" format="date">
  Account expiration date (must be in the future)
</ParamField>

**Example Request:**

```json
{
  "first_name": "John",
  "last_name": "Doe",
  "email": "john.doe@studio.com",
  "password": "securePassword123",
  "role": "user",
  "departments": ["animation-dept-id"],
  "timezone": "America/Los_Angeles"
}
```

**Response:**

```json
{
  "id": "person-uuid",
  "first_name": "John",
  "last_name": "Doe",
  "email": "john.doe@studio.com",
  "role": "user",
  "active": true,
  "departments": ["animation-dept-id"],
  "created_at": "2024-01-15T10:30:00Z",
  "updated_at": "2024-01-15T10:30:00Z"
}
```
